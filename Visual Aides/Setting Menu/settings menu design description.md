# CyphrWhispr Settings Menu Design Description

## Overview
This document describes the full implementation of the CyphrWhispr settings UI, including layout, styling, and behavior.

---

## Window
- Size: 1160 x 960
- Corner radius: 28px
- Background: gradient (#191A20 → #0B0B0D → #050506)
- Border: 1px rgba(255,255,255,0.18)

---

## Tabs
- Three tabs: Shortcut, Models, About
- Active tab uses violet glow and gradient
- Inactive tabs use muted white

---

## Shared Card
- Glass card with blur and subtle border
- Radius: 20px

---

## Page 1: Shortcut
- Dictation hotkey section
- Large shortcut input field with glow
- Tips section
- Reset button

---

## Page 2: Models
- Hardware detection banner
- List of models with selection state
- Scrollable list
- Import + Reveal buttons

---

## Page 3: About
- App icon + title
- Description text
- Feature list
- Footer with version + buttons

---

## Motion
- Subtle transitions
- Soft violet glow
- No aggressive animations

---

## Design Philosophy
- Private
- Local
- Minimal
- Apple-like polish
